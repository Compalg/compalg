Operador para Texto

Operadores disponíveis para texto:

+    Concatenação de texto

Obs: Concatenar significa, ligar ou relacionar. Pode ser entendida também como junção de duas ou mais partes.

Descrição:

Operadores que utilizam variáveis ou valores do tipo literal

Exemplo:

inicio
    literal texto1 <- "Linguagem"
    literal texto2 <- "Programação"
    literal textoresultante <- ""
    textoresultante <- texto1 + " de " + texto2
    escreva textoresultante + " é diversão"
fimalgoritmo
 

Resultado:

Linguagem de Programação é diversão
