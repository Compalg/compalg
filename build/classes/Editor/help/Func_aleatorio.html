Aleatorio

Sintax

real aleatorio()

Descrição

retorna um número real entre 0 e 1

Exemplo

inicio
    inteiro numero , contador
    escrever "chave de totoloto :"
    para contador de 1 ate 6
        numero <- aleatorio ( ) * 47 + 1
        escrever numero , "\t"
    proximo
    fim
