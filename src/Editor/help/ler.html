Leia ou Receba

Sintaxe:

leia (variável)
leia (variavel1, variavel2, ..., variavelN)

receba (variável)
receba (variavel1, variavel2, ..., variavelN)

Descrição:

Solicita a inserção de valores para as variáveis através da consola.
É possível definir a leitura de várias variáveis, desde que estas estejam separadas por virgulas (,).
A leitura de uma variável deve ser precedida da sua declaração.
O valor inserido para cada variável deve ser compatível com o tipo definido na declaração.
 

Exemplo:

/*ALGORITMO: "Leia"
--AUTOR: Bilabila
--DATA: Jan 01 21:58:54 GMT 2012*/
inicio
   inteiro idade
   escreva "Digite um valor:"
   leia ( idade )
   escreva "Você digitou: " , idade

   escreva "Você digitou: " , idade
