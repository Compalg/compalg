Variáveis

Sintaxe:

[tipo] [nome] <- [expressão]
[tipo] [nome] <- [valor] ,[nome] <- [expressão]
[tipo] [nome] <- [expressão]
[tipo] [nome]

Definição:

Define uma variável cujo o valor pode ser alterado durante o tempo de execução.

A declaração de variaveis segue as seguintes regras:

1- O nome tem de começar por uma letra ou pelo carácter  underscore ( _ )
2- O nome não pode ser uma palavra reservada da linguagem.
3- O valor de inicialização tem de ser compativel com tipo de variável definido.
4- Se o valor de inicialização for omitido a variavel é inicializada com os valores por defeito
5- É possivel definir mais que uma variavel utilizando o caracter virgula (,)

Exemplos:

inteiro idade <- 24
real peso <- 20.14
logico repetente
literal nome <- "Augusto" , apelido <- "Bilabila"
caracter sexo
