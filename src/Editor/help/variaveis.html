Simbolos Permitidos

"abcdefghijklmnopqrstuvxywz"
"ABCDEFGHIJKLMNOPQRSTUVXYWZ"
"_0123456789"

LITERAL - começa e termina com aspas " "

exemplo:
literal nome <- "BILA"
_____________________________________________

CARACTER - começa e termina com aspas " "
tem um e um só carecter

exemplo:
caracter sexo <- "F"
______________________________________________

LOGICO - pode ter os valores VERDADEIRO e FALSO

exemplo:
logico repetente <- falso
______________________________________________

INTEIRO - pode ter os valores inteiro

exemplo:
inteiro idade <- 897
______________________________________________

REAL - pode ter os valores reais no formato xx.yy

exemplo:
real media <- 8.5
