Procedimento

Sintaxe:

procedimento nome ( tipo args )
   //comando
fimprocedimento

Descrição:

Os procedimentos são pequenos algoritmos ou sub-algoritmo, que realizam pequenas tarefas e que normalmente são

chamadas ou utilizadas no programa principal (entre o inicio e fim).

Exemplo:

/*ALGORITMO: Uso de Procedimentos
--AUTOR: Bilabila
--DATA E HORA: 7 de Fevereiro de 2012 | 13:26:40 */

procedimento idade ( inteiro idade )
   se ( idade > 18 ) entao
      mostre ( "Adulto " ) , idade
   senao
      mostre ( "Criança" ) , idade
   fimse
fimprocedimento
inicio
   escreva ( "Uso de procedimento " )
   mostre ( "\n--------------------------------\n" )
   escreva "Digite a sua idade:"
   inteiro a
   leia a
   idade ( a )
fimalgoritmo
