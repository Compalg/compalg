Linguagem

Todos algoritmos são escritos em português, e os comandos têm as seguintes características:

1 - A linguagem não á sensível á capitulação ( não faz distinção entre letras minúsculas e maiúsculas);
2 - Todos os algoritmos começam por inicio e terminam com fimalgoritmo (válido somente se não
     utilizar registo);
3 - Em cada linha só pode conter apenas um comando;
4 - O sinal de atribuição é a seta ( <- );
5 - A declaração de variáveis é feita em qualquer local do algoritmo, desde que seja antes de utilizar;
6 - A declaração de registos é feita apenas fora do algoritmo ou seja antes do inicio;

Sintaxe:

inicio
  comandos
fimalgoritmo

registo nome
  variáveis
fimregisto

inicio
  comandos
fimalgoritmo

Esta segunda alternativa da sintaxe, é válido apenas quando utilizamos o registo de dados.

Sugestão de pesquisa:

registo
