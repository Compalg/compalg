Repete

Sintaxe:

repita
    [instruções]
até[condição]
 

Descrição:

Executa as instruções até que a condição seja verdadeira, ou seja:
Enquanto a condição for falsa ela executa as instruções, quando for verdadeira o ciclo pára.

Exemplo:

/*ALGORITMO: Nota Válida
--AUTOR: Bilabila
--DATA: Mon Jan 16 09:48:27 GMT 2012*/
inicio
   real nota
   repita
      escreva ( "Digite uma nota: " )
      leia ( nota )
   ate ( ( nota >= 0 ) e ( nota <= 20 ) )
   escreva "Agora você colocou a nota certa! " , nota
fimalgoritmo

Resultado:

Digite uma nota: 78
Digite uma nota: 25
Digite uma nota: 15
Agora você colocou a nota certa! 15.0
