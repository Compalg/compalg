Tipos de dados Estruturados

Sintaxe:

<tipo dados> nome[dimensão]
<constante> <tipo dados> nome[dimensão]
<tipo dados> nome[dim1][dim2]. . . [dimN]
<constante> <tipo dados> nome[dim1][dim2]. . . [dimN]
<tipo dados> nome[dimensão] <- {valor1, valor2, . . . , valorN}
<constante> <tipo dados> nome[dimensão] <- {valor1, valor2, . . . , valorN}

Estruturas de dados homogéneas (Arrays):

Define um conjunto de variáveis do mesmo tipo de dados, que podem ser acedidos através de um índice.

Regras:

1- A definição do nome obedeçe ás mesmas regras das variáveis ;
2- A dimensão obrigatóriamente tem de ser um valor inteiro ou uma expressão com resultado inteiro;
3- A dimensão ou o tamanho do array deve ser colocada dentro de parênteses rectos;
5- São permitidas tantas dimensões quantas as desejadas.
6- Se o valor de inicialização for omitido, então o array é inicializado com os valores por defeito
7- A inicialização é feita com os dados compreendidos entre chavetas e separados por virgulas
8- Se o número de valores for insuficiente os últimos elementos são incializado com os valores por defeito

Nota: A primeira posição de um array é sempre 0 e a última é a dimensão - 1, ou seja se definir o a dimensão
10 para um array qualquer, então a primeira posição é 0 e a última é 9.

Exemplo:

constante literal sexo[2]<-{"Masculino", "Femenino"}
inteiro vari1 [ 2 ] [ 4 ] [ 1 ] [ 1 ] // array vari1 com oito elementos.
real nota [ 10 ] // array nota com 10 elementos.
